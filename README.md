# CovidJabar app built using ReactNative

Learning some new react features by making this simple app. Feel free to clone this repo.

All data is obtained from `https://covid19-public.digitalservice.id/api/v1/`

## Tech Stacks:
- React Native
- Reack Hooks/Context
- Map
